list_alt Detailed Findings

Critical Source: AWS API Gateway Tag: A01:2021

Unauthenticated User Profile Access

API endpoint `/api/v1/user/details` allows access to private user metadata without a valid JWT token. Potential for massive data scraping.

lightbulb

Recommendation

Implement Mandatory Access Control (MAC) on all user detail endpoints and verify token identity.

High Source: Customer Database Tag: A03:2021

SQL Injection in Search Query

The search bar input is not properly sanitized before being passed to the SQL engine. Allows for blind SQL injection.

lightbulb

Recommendation

Use parameterized queries or an ORM for all database interactions involving user input.

High Source: S3 Bucket Config Tag: A05:2021

Publicly Accessible Storage Bucket

Bucket `siarc-prod-assets` is currently set to public read access. Contains internal build artifacts.

lightbulb

Recommendation

Enable "Block Public Access" and use CloudFront signed URLs for asset distribution.

Medium Source: Docker Image Tag: A06:2021

Outdated Library Version

Production container is running `libssl 1.1.1f` which has 2 known low-severity CVEs.

lightbulb

Recommendation

Update base image to the latest stable Debian/Alpine version and rebuild containers.

Critical Source: Auth Service Tag: A07:2021

Hardcoded API Credentials

Production credentials for the Stripe API were found in plain text within the application's configuration file.

lightbulb

Recommendation

Migrate all secrets to AWS Secrets Manager or HashiCorp Vault immediately.

Medium Source: Nginx Config Tag: A05:2021

Missing HSTS Headers

HTTP Strict Transport Security is not enabled on the production load balancer.

lightbulb

Recommendation

Add `Strict-Transport-Security` header with a long max-age and include subdomains.
